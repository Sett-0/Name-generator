# Name generator 
## About
A fairly simple algorithm for generating fancy words.
## How to use
```shell
usage: gen.py [-h] [--word_len WORD_LEN] [--num_words NUM_WORDS] [--num_tries NUM_TRIES]

options:
  -h, --help            show this help message and exit
  --word_len WORD_LEN   Word length (min, max). Ex. (4,8), (4,4). Default = (3,7)
  --num_words NUM_WORDS Number of words in a line. Default = 1
  --num_tries NUM_TRIES Number of lines to generate. Default = 15
```
